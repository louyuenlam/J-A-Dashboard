📋 How to Auto-Sync Your Google Calendar

This dashboard automatically extracts staff hours, bookings, and leave from your Google Calendar. Here's how it works:

🔑 Event Naming Conventions

Use staff names in your calendar event titles to automatically categorize them:

Your Staff List:
• Fulltime: Amus, Jimmy (work Mon-Sat, only enter days OFF)
• Parttime: Mi, Toby, Emily (enter all working shifts)
💡 To add/edit staff: Edit the staffList in the code (around line 1058)
Fulltime Staff (Amus & Jimmy) - Days OFF:
• Amus - Off or Amus Holiday
• Jimmy - Leave or Jimmy Vacation
System assumes they work Mon-Sat and calculates hours minus days off
Parttime Staff (Mi, Toby, Emily) - Working Shifts:
• Mi - Working 12-3pm
• Toby Shift - Studio
• Emily - Film Lab 2-6pm
Hours calculated from calendar event duration
Bookings (Photoshoots):
• Photoshoot - Sarah Johnson Graduation
• Wedding - Emma & David
• Family Portrait - The Smiths
Keywords: photoshoot, wedding, portrait, graduation, couple, family, maternity
Film Lab Processing:
• Film Processing - John Davies 3 rolls
• Film Lab - VIP Member 120 B&W
• Develop - Lisa Chen ECN-2
Keywords: film, processing, develop, lab, scan
Annual Leave (Fulltime staff only):
• Amus - Holiday or Amus Leave
• Jimmy - Annual Leave or Jimmy Off
• Amus Vacation - Spain
Each fulltime staff has 28 days annual leave. Parttime staff don't have leave tracking.

⚙️ API Setup (Required)

  1. Go to Google Cloud Console
  2. Create a new project: "J&A Studio Dashboard"
  3. Enable "Google Calendar API"
  4. Create credentials:
    • API Key
    • OAuth 2.0 Client ID (Web application)
  5. Add to authorized JavaScript origins: http://localhost
  6. Copy your Client ID and API Key
  7. Edit this HTML file and replace:
    • Line ~1050: CLIENT_ID
    • Line ~1051: API_KEY
  8. Save and reload this page
  9. Click "Connect Calendar" button

✅ What Gets Auto-Calculated

  • ✓ Staff working hours (week/month/year)
  • ✓ Annual leave days taken and remaining
  • ✓ Upcoming photoshoot bookings
  • ✓ Film lab processing queue
  • ✓ Weekly hours breakdown per staff
  • ✓ Progress toward weekly targets
💡 Pro Tip: Use consistent naming in your Google Calendar and this dashboard will automatically organize everything for you!
